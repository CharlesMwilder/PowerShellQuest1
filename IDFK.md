# PowerShellQuest1
## les commandes unix et les commandes PowerShell équivalente

- cp  >  Copy-Item
- rm  > Remove-Item
- cd  > Set-Location
- mkdir  > New-Item
- man  > Get-Help
- history  > Get-History
- alias  > Set-Alias
- cat  > Get-Content


## les lignes de commandes utilisées pour trouver les cmdlets PowerShell

    Get-help copy                                                                                                       
    Get-Help Copy-Item                                                                                                     
    Get-Help mkdir
    Get-Help cat
